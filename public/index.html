Brasileirão Dataset API


Bem-vindo(a) à documentação da API Brasileirão Dataset. Aqui você encontrará informações dos jogos do Campeonato Brasileiro Série A desde o seu início em 2003 até 2023.
Todos os dados foram obtidos do repositório de @adaoduque no GitHub.

Métodos Getters


Buscar Todos os Jogos

Endpoint: /api/jogos
Descrição: Recupera todos os jogos registrados na API.

Buscar Jogo por ID

Endpoint: /api/jogos/:id
Parâmetros:

  • id: ID do jogo que você deseja buscar (obrigatório)

Descrição: Recupera um jogo específico pelo ID.

Buscar Jogos por Intervalo de Datas

Endpoint: /api/jogos/data/:start/:end
Parâmetros:

  • start: Data de início no formato YYYY-MM-DD (obrigatória)
  • end: Data de fim no formato YYYY-MM-DD (obrigatória)

Descrição: Recupera todos os jogos dentro do intervalo de datas fornecido.

Buscar Jogos por Confronto e Intervalo de Datas

Endpoint: /api/jogos/confronto/:time1/:time2/:datainicio?/:datafim?
Parâmetros:

  • mandante: Nome do primeiro time (obrigatório)
  • visitante: Nome do segundo time (obrigatório)
  • datainicio: Data de início no formato YYYY-MM-DD (opcional)
  • datafim: Data de fim no formato YYYY-MM-DD (opcional)

Descrição: Recupera todos os jogos entre os times fornecidos e, se definido, dentro do intervalo de datas.

Buscar Jogos por Time e Intervalo de Datas

Endpoint: /api/jogos/time/:time/:datainicio?/:datafim??
Parâmetros:

  • mandante: Nome do time (obrigatório)
  • datainicio: Data de início no formato YYYY-MM-DD (opcional)
  • datafim: Data de fim no formato YYYY-MM-DD (opcional)

Descrição: Recupera todos os jogos para um time e, se definido, dentro do intervalo de datas.
